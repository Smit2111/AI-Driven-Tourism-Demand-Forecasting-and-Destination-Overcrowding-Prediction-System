# AI-Driven-Tourism-Demand-Forecasting-and-Destination-Overcrowding-Prediction-System

An end-to-end machine learning web application that forecasts tourism demand, predicts overcrowding risk, and delivers smart travel recommendations across India's top destinations — built with Streamlit, Scikit-learn, and Plotly.
🚀 Features

🔮 AI Travel Intelligence — Rule-based seasonal demand engine that predicts peak/off-season risk for destinations based on Winter, Summer, Monsoon, and Festive patterns
🤖 ML Demand Forecaster — Random Forest Regressor trained on visitor count data; predicts crowd levels relative to each place's historical average
📊 BI Dashboard — Interactive Plotly charts, correlation heatmaps, and state-wise tourism analytics for data-driven decision making
💬 AI Chatbot — Keyword-aware travel assistant for destination advice, budgeting, safety tips, and itinerary planning
💰 Budget Trip Planner — Auto-generates day-wise itineraries with smart budget breakdowns (stay, transport, food, activities)
🔍 Place Search & Comparison — Search and compare destinations by rating, crowd level, ticket price, and best season
🗺 Map Explorer — State-wise interactive map of tourist hotspots with top-rated place listings
🚨 Overcrowding Alerts — Flags high-risk states and generates crowd management strategies for travel agents
🔐 Role-based Auth — Separate dashboards for Travellers and Travel Agents, with CSV-based user management and profile editing
